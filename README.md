# OpenClaw Workspace

Agent workspace for OpenClaw. Contains persona files, skills, memory, and project code.

## Contents

- **AGENTS.md**, **SOUL.md**, **IDENTITY.md** – Agent persona and instructions
- **memory/** – Daily memory logs
- **MEMORY.md** – Curated long-term memory
- **skills/** – Custom OpenClaw skills (farcaster, nft, cryptoart, etc.)
- **config/** – Template/bootstrap configs
- **api/**, **web/** – Project code

## Setup

1. Clone to `~/.openclaw/workspace` (or your configured workspace path)
2. Run `openclaw setup --workspace <path>` to seed any missing files

## Privacy

Keep this repo **private**. It contains agent memory and preferences.
