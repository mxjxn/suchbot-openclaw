# TOOLS.md - Local Notes

Skills define _how_ tools work. This file is for _your_ specifics.

## /cryptoart Channel

- Post: `fc_cast.sh --channel cryptoart` or `fc_thread.sh --channel cryptoart`
- Channel feed: `fc_feed.sh --channel cryptoart`

## Artist Lookup Flow

Subgraph listings → seller addresses → `fc_user.sh --address <addr>` → `fc_feed.sh --fid <fid>` for cast history.

## Research Docs

Artist research outputs go in `research/artists/`.

## Platform Formatting

- **Discord/WhatsApp:** No markdown tables — use bullet lists
- **Discord links:** Wrap in `<>` to suppress embeds: `<https://example.com>`
- **WhatsApp:** No headers — use **bold** or CAPS

## What Goes Here

Camera names, SSH hosts, TTS voices, device nicknames, etc.
