# Cast Link Format

**Correct format for suchbot casts:**
https://farcaster.xyz/suchbot/0x{hash}

**Incorrect formats to avoid:**
- https://warpcast.com/cast/0x{hash} (warpcast links, not farcaster.xyz)
- Any other format

**Notes:**
- always use farcaster.xyz for suchbot links
- hash is the 32-char hex after 0x prefix
